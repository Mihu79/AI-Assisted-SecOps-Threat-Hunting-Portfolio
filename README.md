# 🛡️ AI-Assisted SecOps & Threat Hunting Portfolio
### Project: Askeal AI Beta Testing & Validation

Este un depozit de cazuri de utilizare (Use Cases) și scenarii de investigație realizate în cadrul programului de Private Beta al asistentului AI **Askeal**. Obiectivul proiectului este validarea preciziei AI în operațiuni de securitate (SecOps) și generarea de artefacte de detecție (KQL, PowerShell, Playbooks).

## 🛠️ Tehnologii și Metode Utilizate
* **Platformă AI:** Askeal (SecOps Assistant)
* **Limbaje de interogare:** KQL (Kusto Query Language), PowerShell, SQL
* **Framework-uri:** MITRE ATT&CK (Reconnaissance, Initial Access, Lateral Movement)
* **Metodologie:** NIST Incident Response Lifecycle

## 📈 Obiective de Învățare
1.  Optimizarea interacțiunii cu LLM-urile prin **Advanced Prompt Engineering** pentru securitate.
2.  Reducerea timpului de răspuns (**MTTR**) prin automatizarea generării de interogări de vânătoare (Threat Hunting).
3.  Validarea tehnică a răspunsurilor AI pentru eliminarea halucinațiilor în context critic.

---
*Acest proiect este realizat în regim de voluntariat/testare pentru dezvoltarea comunității de securitate cibernetică.*
